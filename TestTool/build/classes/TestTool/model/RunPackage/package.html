Run Package

This package contains the classes involved with running the whole GalacTest software. It is comprised of two folders, "model" and "view." The model folder has the two separate classes for the different run types for GalacTest: "StudentRun" and "TeacherRun." The view folder contains the JFrame classes for the respective run type home pages: "JFrameStudentHome" and "JFrameTeacherHome."
